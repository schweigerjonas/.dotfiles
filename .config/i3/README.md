# Configuration for i3 Window Manager

## Prerequisites

### Additional Packages

- [nitrogen](https://github.com/l3ib/nitrogen) - Background browser and setter.
- [bumblebee-status](https://github.com/tobi-wan-kenobi/bumblebee-status) - Modular, theme-able status line generator for i3.
- [flameshot](https://github.com/flameshot-org/flameshot) - Simple screenshot software.
- [pulsectl](https://github.com/mk-fg/python-pulse-control) - Interface and bindings for PulseAudio.
- [rofi](https://github.com/davatorium/rofi) - Application launcher.
